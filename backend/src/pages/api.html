🚀 FactoryTrack API

Sistema de gerenciamento de pedidos, clientes e produtos

📖 Sobre o Projeto

A FactoryTrack API é um backend desenvolvido em Node.js com SQLite, projetado para gerenciar operações de uma pizzaria ou sistema de pedidos.

Ela permite:

  • 👤 Gerenciamento de usuários (admin e atendente)
  • ⚙️ Cadastro de produtos
  • 📦 Controle de pedidos
  • 📞 Cadastro de clientes
  • 📊 Monitoramento completo via rota de debug

Tecnologias:

Node.js Express SQLite JWT

🔐 Autenticação

Utiliza JWT (JSON Web Token).

Após login, envie o token no header:

Authorization: Bearer SEU_TOKEN_AQUI

📦 Estrutura de Dados

Cliente:

{ "id": 1, "nome": "João", "telefone": "11999999999", "endereco": "Rua A, 123" }

Produto:

{ "id": 1, "nome": "Pizza Calabresa", "descricao": "Deliciosa", "disponivel": true }

Ordem:

{ "id": 1, "cliente": {...}, "itens": [ { "produto": 1, "quantidade": 2 } ], "status": "recebido" }

📡 Endpoints

POST /api/auth/login

Realiza login e retorna token JWT

{ "email": "admin@email.com", "senha": "123456" }
GET /api/produtos

Lista todos os produtos

POST /api/produtos

Cria um produto

{ "nome": "Pizza Calabresa" }
GET /api/clientes

Lista clientes

POST /api/clientes

Cria cliente

{ "nome": "João", "telefone": "11999999999", "endereco": "Rua A, 123" }
GET /api/ordens

Lista pedidos

POST /api/ordens

Cria um novo pedido

{ "cliente": 1, "itens": [ { "produto": 1, "quantidade": 2 } ] }
PATCH /api/ordens/:id/status

Atualiza status do pedido

{ "status": "em_preparo" }
GET /api/debug

Retorna todos os dados do sistema
